# Fretboard Adventures

Some interactive flash-card style games for learning guitar.

## Some useful dev commands

1. `npm install`
1. `npm run dev`
1. `npm run test:watch`
1. `npm run check`
   1. Run this before pushing

## Install on iPhone

1. Open the site in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (square with arrow)
3. Select **Add to Home Screen**
4. The app will launch fullscreen without browser controls
